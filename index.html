Projects

Fused Quantized Attention Kernel
2024.10 - 2024.12 Parallel Computing

CUDA implementation of optimized attention mechanisms with block-wise quantization support (MXFP4, NF4, and NVFP4). Incrementally fused the naive dequantization-projection-attention sequence into custom kernels to minimize memory overhead.

SKiP: SVM with K-nearest neighbor and Probabilistic weighting
2024.11 - 2024.12 Machine Learning

Novel SVM variant that assigns adaptive sample weights based on K-Nearest-Neighbors and class probability distributions. Designed to reduce the impact of outliers and boundary noise in classification tasks across multiple datasets.

Pintos
2024.03 - 2024.06 Operating System

Implementation of core operating system features including thread scheduling, synchronization primitives, alarm clock, priority scheduling, and advanced scheduler. Built on the Pintos framework for x86 architecture.

Traveling Salesman Problem Solver
2024.06 Algorithm

Comprehensive implementation of TSP algorithms including MST-based 2-approximation, Held-Karp dynamic programming, greedy heuristics (node-based, edge-based), and a custom hybrid algorithm combining edge and node greedy approaches. Benchmarked across five datasets: the custom-made jam20 dataset, a280 from TSPLIB, and three instances sourced from TSP Test Data (TTD), including XQL662, Kazakhstan, and the Mona Lisa TSP Challenge.

Efficient Similarity Search via Adaptive Product Quantization
2024.04 - 2024.05 Data Mining

Comparative evaluation of Product Quantization (PQ) frameworks including k-means variants (standard, k-means++, mini-batch, bisecting) , Optimized PQ (OPQ). Proposed a custom "Adaptive Product Quantization (APQ)" algorithm utilizing Pearson correlation-based dimension partitioning for efficient similarity search in resource-constrained environments.

Sorting Algorithms Analysis
2024.04 Algorithm

Comprehensive C++ implementation and performance analysis of 12 sorting algorithms: Merge Sort, Heap Sort, Quick Sort, Intro Sort, Tim Sort, Bubble Sort, Insertion Sort, Selection Sort, Cocktail Shaker Sort, Comb Sort, Library Sort, and Tournament Sort. Evaluated using datasets of sizes n (1K to 1M) across five distributions: Random, Dense Random (high-frequency duplicates), Ascending, Descending, and Partially Sorted.
